# IoskeleyMono

A free, open-source alternative to Berkeley Mono, built by configuring the [Iosevka](https://github.com/be5invis/Iosevka) typeface to match Berkeley Mono's look and feel.

**Author:** Ahmed Hatem ([@ahatem](https://github.com/ahatem))
**Source:** https://github.com/ahatem/IoskeleyMono
**Version:** v2.0.0-beta.1
**License:** SIL Open Font License 1.1

## Variants

- **Normal** — standard width
- **SemiCondensed** — slightly narrower
- **Condensed** — narrow

## Weights

Each variant includes 10 weights (Thin, ExtraLight, Light, SemiLight, Regular, Medium, SemiBold, Bold, ExtraBold, Black) in regular and italic styles — 20 TTF files per variant.

The files here are the **Unhinted** TTF builds, suitable for macOS, Linux, and modern Windows rendering.

NerdFont variants (with icon support) and web fonts (WOFF2) are available from the [releases page](https://github.com/ahatem/IoskeleyMono/releases/tag/v2.0.0-beta.1).
