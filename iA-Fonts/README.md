# iA Fonts

Free, open-source writing fonts from [iA](https://ia.net), bundled with [iA Writer](https://ia.net/writer/). Based on a modification of [IBM Plex](https://github.com/IBM/type).

**Author:** Information Architects Inc. ([@iaolo](https://github.com/iaolo))
**Source:** https://github.com/iaolo/iA-Fonts
**License:** SIL Open Font License 1.1

## Families

- **iA Writer Mono** — Monospaced writing font; each character occupies exactly one column
- **iA Writer Duo** — "Duospace" writing font; most characters are monospaced, but wider letters (m, w) take up 2 column widths for improved readability
- **iA Writer Quattro** — Proportional writing font; character widths adapt naturally, optimized for longer reading and writing sessions

## Weights

Each family includes 4 static TTF files: Regular, Italic, Bold, BoldItalic.

Variable font versions (supporting a continuous weight/width axis) and webfonts (WOFF2) are available from the [source repository](https://github.com/iaolo/iA-Fonts).
